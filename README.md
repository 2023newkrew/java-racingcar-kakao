# 문자열 계산기 
### 구현할 기능 목록
- [x] 문자열을 분리 후 더하는 기능
  - [x] 문자열을 숫자 리스트로 변환하는 기능
    - [x] 문자열을 split하는 기능
    - [x] 문자열을 숫자로 변환하는 기능

  - [x] 분리된 숫자들을 더해서 출력하는 기능
- [x] 커스텀 구분자를 추가하는 기능
  - [x] 문자열에서 커스텀 구분자를 추출하는 기능
- [x] 문자열 입출력 기능
- [x] 계산기 Controller
