# java-racingcar-kakao
* 자동차 이름 입력 받는 기능
* 이동 횟수 입력 받는 기능
* 자동차 이름과 전진여부 출력
* 입력받은 자동차 이름 쉼표 구분자로 split
* 자동차 이름길이 검증
* 0~9 사이 정수 random값 생성
  * 전전 여부 결정
* 최대 전진 거리 판별
  * 우승자 모두 출력

# string-add-calculator
* 커스텀 구분자 존재 여부 확인
    * 커스텀 구분자 제외한 숫자로 이루어진 문자열 추출
    * 커스텀 구분자 추출

* 숫자 이외의 값/음수인 경우 예외처리
